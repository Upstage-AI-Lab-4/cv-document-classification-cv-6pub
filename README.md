## 2. 채택한 개발 기술

### Pytorch, torchvision, scikit-learn

- Pytorch 프레임워크
  - 딥러닝 파이썬 프레임워크
    
- torchvision, pillow
  - 컴퓨터비전용 데이터셋, 모델, 이미지 전처리 제공
  - 이미지 라이브러리
    
- scikit-learn
  - 성능 평가 지표
    
### effdet, efficientnet_pytorch

- effdet
  - efficientDet 객체 탐지 모델 라이브러리
    
- efficientnet_pytorch
  - efficientNet 분류 모델 라이브러리
